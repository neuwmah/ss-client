# SS Client

Automatic use of healing potions from hotbar by memory hook.

> No bypass included.<br>
> Only working in private servers.<br>
> Application published on March 13, 2026.

[discord.gg/mixmaster](https://discord.gg/mixmaster)

## Features

- Auto use of healing potions HP/MP from hotkeys F1~F10.<br>
- Auto detect potion keys when toggled on.<br>
- Auto detect lowest HP/MP allies before potion usage.<br>
- Auto replace default chat messages with HP/MP key log.<br>
- Auto pause healing potions when skill is active.<br>
- Auto stop healing potions when hero is dead.<br>
- Auto detect current game window before activation.

*Press DEL key to toggle ON/OFF*
